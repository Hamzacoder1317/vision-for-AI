Meta releases big, new open source AI large 'language model'

Meta, formerly known as Facebook, has launched a commercial version of Llama-v2, which is an open-source large language model (LLM) powered by artificial intelligence (AI). This advanced language model has the capability to generate text, images, and code using its sophisticated AI technology.

Initially, the first version of the Large Language Model Meta AI (Llama) was exclusively accessible to approved researchers and organizations. However, the situation changed in early March when the model was leaked online, making it available for download and usage to anyone.

In response to the unauthorized release, Meta took measures to remove the leaked program from various platforms, such as GitHub and the open-source AI group Hugging Face. Despite these efforts, the code had already spread widely across the internet, making it challenging to control its distribution. As a result, Meta eventually decided to embrace the situation and officially released the Llama model to the public.
